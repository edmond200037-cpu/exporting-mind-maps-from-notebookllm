# NotebookLM 心智圖導出器 v5.3.0

## 🎯 快速開始

### 安裝步驟
1. 打開 Chrome 瀏覽器
2. 進入 `chrome://extensions/`
3. 開啟右上角的「開發者模式」
4. 點擊「載入未封裝項目」
5. 選擇 `extension_fixed` 資料夾
6. 看到 🧪 圖標表示安裝成功

### 使用方法
1. 打開 NotebookLM 心智圖頁面
2. 點擊瀏覽器工具列的 🧪 圖標
3. 點擊「🧪 檢測心智圖」
4. 選擇導出格式
5. 點擊「💾 導出心智圖」

## 🔧 修復內容

| 問題 | 原版本 | 測試版 | 改善效果 |
|------|--------|--------|----------|
| 超時限制 | 25秒 | 60秒 | 支持大型心智圖 |
| 連接檢測 | 25px容錯 | 50px容錯 | 提高檢測準確性 |
| 層級分組 | 5px容錯 | 15px容錯 | 減少錯誤分組 |
| 節點過濾 | 嚴格過濾 | 寬鬆過濾 | 保留更多有效節點 |
| 中文支持 | 基本支持 | 完整支持 | 更好的中文處理 |

## 📋 文件說明

- `manifest.json` - 擴展配置文件
- `content.js` - 核心提取邏輯 (已修復)
- `popup.html/js` - 用戶界面 (中文化)
- `background.js` - 後台服務
- `FIXES.md` - 技術修復詳情
- `測試說明.md` - 詳細測試指南
